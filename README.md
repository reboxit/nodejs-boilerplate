# This is Jarvis!

Jarvis is the backend of our app Reboxit

## How to install


```
git clone https://github.com/reboxit/nodejs-boilerplate.git
```

## How to run the app

#### On your local machine



```
cp .env.sample .env
npm run start:dev
```

#### On docker

```
docker-compose up --build api
```
